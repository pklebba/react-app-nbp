# NBP API Currencies

## Requirements

To use this project you must have installed Node.JS in version 6 or greeter.

---

## Install

    $ git clone https://github.com/pklebba/react-app-nbp
    $ cd react-app-nbp
    $ npm install # you can use "yarn install" if you wish

## Start & watch

    $ npm start

##  Build for production

    $ npm run build
#### If using yarn
    $ yarn build

## Testing
    
    $ npm run test
#### If using yarn    
    $ yarn test
